# LLVM

Introductory programming in raw LLVM assembler.

# EXAMPLE

```
$ make
llvm-as fizzy.ll
llc fizzy.bc
mkdir -p bin/
gcc -o bin/fizzy fizzy.s
bin/fizzy
1
2

4
Buzz
Fizz

...
98
Fizz
Buzz
```

# REQUIREMENTS

* [coreutils](https://www.gnu.org/software/coreutils/coreutils.html)
* [LLVM](http://llvm.org/)
* [gcc](https://gcc.gnu.org)
* [make](https://www.gnu.org/software/make/)
