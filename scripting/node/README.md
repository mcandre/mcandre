# toys/scripting/node

Introductory Node.js programs

# EXAMPLE

```console
$ cd node-hello
$ node lib/hello.js
Server running at http://127.0.0.1:8124/
$ open http://127.0.0.1:8124/
...
```

# REQUIREMENTS

* [Node.js](http://nodejs.org/) 14.15.1+

# DEVELOPMENT

## Lint

Keep the code tidy:

```console
$ grunt lint
```

## Document

```console
$ grunt doc
```
